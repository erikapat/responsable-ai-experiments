# Responsible AI Experiments

Repositorio para experimentos independientes relacionados con Responsible AI.

## Experimentos

- `experiments/image_bias`: experimento existente sobre sesgos en generación de imágenes.
- `experiments/time_series_reweighting`: espacio reservado para el futuro experimento de reweighting de series temporales.

Cada experimento conserva sus propios datos, código, dependencias y resultados. Los secretos deben configurarse en archivos `.env` locales; solo los archivos `.env.example` sin valores reales se versionan.
